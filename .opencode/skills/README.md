# Skills

Specialized skills that the agent can use for specific tasks.

## Available Skills

| Skill | Description |
|-------|-------------|
| **code-agent** | Debug, implement, edit code with context awareness |
| **analysis-agent** | Code review, evaluation, and quality assessment |
| **research-agent** | Investigation and information gathering |
| **git-commit** | Conventional commit message generation |
| **pr** | Pull request description and title writing |
| **rlm** | Recursive context management for large tasks |
| **self-learning** | Self-improvement and pattern recognition |
| **code-review** | Structured code review workflow with priority-based classification (CRITICAL/IMPORTANT/SUGGESTION) |
| **prompt-engineer** | Prompt engineering best practices, safety frameworks, bias mitigation, and responsible AI usage |
| **security-audit** | OWASP-based security audit workflow |

## Usage

Skills are automatically loaded based on the detected intent and task requirements.

See the parent [README](../../../README.md) for more information on how the system works.
