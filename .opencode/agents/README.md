# Agents

Subagents for complex tasks and specialized workflows.

## Available Agents

| Agent | Description |
|-------|-------------|
| **intent-director** | Intent classification and task routing |
| **rlm-subcall** | Large context chunking and recursive processing |

## Usage

Agents are invoked by the main agent for complex tasks that require specialized processing or multi-step workflows.

See the parent [README](../../../README.md) for more information on how the system works.
